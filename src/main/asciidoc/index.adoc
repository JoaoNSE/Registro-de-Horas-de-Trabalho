== Introdução
Essa é a documentação da API REST para o sistema de Registro de Horas de Trabalho. Aqui consta a explicação de uso
e funcionamento de todos os endpoints desta API.

== API dos Registros de Horas (HoursRecord)
Os endpoints que compõe o CRUD para manipular os registros de horas de trabalho (HoursRecord).

=== Get Record By User Id
Obtains a specific person registered in the application by its unique identifier.
Obtém todos os registros de horas cadastrados de um usuário indicado pelo indentificador (Id).
